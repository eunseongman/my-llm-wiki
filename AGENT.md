# AGENT.md

이 파일은 사람을 위한 짧은 안내다.
실제 세션 지침은 루트의 `AGENTS.md`를 우선 참고한다.

핵심 규칙만 요약하면 다음과 같다.

- 이 저장소는 앱 프로젝트가 아니라 개인 지식 도서관이다.
- `정리해줘`라고 하면 현재 대화를 문서로 반영해야 한다.
- `정리해줘`는 채팅 요약 요청이 아니라 파일 반영 요청이다.
- 기본 산출물은 `report/` 아래의 읽는 글이다.
- 최소 산출물은 `report + session + log`다.
- `library/` 문서는 필요한 핵심 개념만 선택적으로 만든다.
- 기존 문서가 있으면 새 문서를 남발하지 말고 보강한다.
- 파일 반영이 끝나기 전에는 말로만 정리하고 끝내면 안 된다.
