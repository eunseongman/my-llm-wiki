# LLM 위키

## 한 줄 정의

- LLM이 문서를 단순 검색하는 대신, 읽고 정리하고 갱신하며 유지하는 지속적 지식 문서 집합이다.

## 설명

- 전통적인 RAG는 질문할 때마다 원문에서 다시 찾고 다시 조합한다.
- LLM 위키는 질문 전에 이미 정리된 지식 문서를 만들어 둔다.
- 새 대화나 새 자료가 들어오면 기존 문서를 업데이트하면서 지식이 축적된다.

## 왜 중요한가

- 같은 질문을 다시 했을 때도 매번 처음부터 재구성하지 않아도 된다.
- 세션을 넘어서는 장기 기억 보조 구조를 만들 수 있다.
- 사용자는 원문보다 더 읽기 쉬운 형태로 자신의 학습 흔적을 복습할 수 있다.

## 예시

- 어떤 개념을 오늘 대화로 배웠다면 session note만 남기는 것이 아니라 concept note와 topic note까지 연결해 둔다.
