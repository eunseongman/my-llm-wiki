# 지식 도서관과 앱의 차이

## 한 줄 정의

- 앱은 기능을 제공하는 산출물이고, 지식 도서관은 읽고 연결하고 축적하는 문서 중심 산출물이다.

## 설명

- 앱 관점에서는 UI, 서버, 기능 흐름이 중심이 된다.
- 지식 도서관 관점에서는 세션, 개념, 주제, 종합, 질문 문서가 중심이 된다.
- 사용자가 이번 대화에서 명확히 원한 것은 "앱을 만드는 것"이 아니라 "이 폴더를 도서관으로 쓰는 것"이었다.

## 왜 중요한가

- 이 차이를 분명히 해야 구현 우선순위가 바뀐다.
- 앱 중심으로 가면 인프라를 먼저 만들게 되지만, 도서관 중심으로 가면 문서 구조와 운영 규칙을 먼저 만들게 된다.
- 저장소의 목적이 흔들리지 않게 해 준다.

## 예시

- `library/index.md`와 `library/log.md`를 갖춘 것은 앱 기능이 아니라 도서관 운영 장치다.
